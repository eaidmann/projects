A simple vending machine application where a user can choose from the list of items available, then insert the item number and the amount of money inserted.

If the customer insert enough money into the machine, the machine will provide the balanced remained for the purchase and return back to customer. If the customer insert insufficient amount of money needed, the machine will prompt for insufficient balance and request for the remaining money need to be inserted from the customer.
